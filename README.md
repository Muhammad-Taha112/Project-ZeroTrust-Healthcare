# ZeroTrust Healthcare

## Overview
ZeroTrust Healthcare is a secure healthcare portal based on the Zero Trust Security Architecture. The system provides secure access to healthcare resources by verifying every user before granting access.

## Features
- Secure Login System
- Multi-Factor Authentication (MFA)
- Role-Based Access Control (RBAC)
- Activity Logging
- Secure User Authentication
- Admin Dashboard
- User Dashboard

## Technologies Used
- HTML5
- CSS3
- JavaScript

  

## Objective
The objective of this project is to improve healthcare data security by implementing Zero Trust Security Architecture and preventing unauthorized access.
